# Common ROS2 Commands

## Build

```bash
colcon build --symlink-install
```

## Source

```bash
source install/setup.bash
```

## Launch Simulation

```bash
ros2 launch ur5e_simulation ur5e_simulation.launch.py
```

## Topics

```bash
ros2 topic list
```

## Nodes

```bash
ros2 node list
```

## Parameters

```bash
ros2 param list
```
